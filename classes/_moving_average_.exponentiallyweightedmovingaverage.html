  • Implementation of MovingAverage.tick

    • Defined in moving-average.ts:182

    Updates the current average by multiplying the alpha value with the difference of the last average and the current average.

    Averages are calculated by dividing the sum through the sampling interval.

    If the alpha value is high the last average has more weight and vice versa.
